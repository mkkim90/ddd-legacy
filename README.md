# 키친포스

## 퀵 스타트

```sh
cd docker
docker compose -p kitchenpos up -d
```

## 요구 사항
### 1. 상품 관리

####  상품 생성
- [ ] 가격은 필수값이며, 0보다 커야한다.
- [ ] 상품명은 필수값이며, 비속어가 포함되지 않아야한다. 

#### 상품 수정
- [ ] 가격은 필수값이며, 0보다 커야한다. 
- [ ] 수정하고자 하는 상품이 존재해야한다.
- [ ] 수정한 가격이 해당 상품이 포함된 메뉴 가격에 반영되어야 한다. 
  - [ ] 메뉴 가격이 상품의 합산 가격보다 클 경우, 해당 상품을 비노출 처리한다.

#### 상품 조회
- [ ] 해당 상품을 전체 조회한다.

---

### 2. 메뉴 관리

#### 메뉴 생성
- [ ] 가격은 필수값이며, 0보다 커야한다. 
- [ ] 생성하려는 하는 메뉴의 그룹이 존재해야한다.
- [ ] 메뉴 상품은 필수값이며, 최소 1개 이상이어야 한다.
- [ ] 메뉴 상품은 반드시 상품 테이블에 존재해야 한다.
- [ ] 메뉴 상품의 수량은 0보다 커야한다.
- [ ] 메뉴 가격이 메뉴 상품들의 합산 가격보다 크지 않아야 한다. 
- [ ] 메뉴명은 필수값이며, 비속어가 포함되지 않아야 한다.

### 메뉴 수정
- [ ] 가격은 필수값이며, 0보다 커야 한다.
- [ ] 수정하려는 메뉴가 존재해야 한다.
- [ ] 수정된 가격이 메뉴 상품들의 가격 합과 일치해야 한다.

### 메뉴 노출
- [ ] 노출하려는 메뉴가 존재해야 한다.
- [ ] 메뉴 가격이 메뉴 상품들의 가격 합과 일치해야 한다.

### 메뉴 비노출
- [ ] 비노출하려는 메뉴가 존재해야 한다.

### 메뉴 조회
- [ ] 모든 메뉴를 조회할 수 있어야 한다.

---

## 3. 메뉴 그룹 관리

### 메뉴 그룹 생성
- [ ] 메뉴 그룹명은 필수값이다.

### 메뉴 그룹 조회
- [ ] 모든 메뉴 그룹을 조회할 수 있어야 한다.

---

## 4. 주문 관리

### 주문 생성
- [ ] 주문 타입은 필수값이다.
- [ ] 메뉴 주문은 필수값이며, 최소 1개 이상이어야 한다.
- [ ] 메뉴 주문의 메뉴가 존재해야 한다.
- [ ] 메뉴 주문 수량은 **포장 또는 배달 주문의 경우** 0보다 커야 한다.
- [ ] 메뉴 주문의 메뉴는 **노출된 메뉴여야 한다**.
- [ ] 메뉴 주문 가격은 메뉴 가격과 동일해야 한다.
- [ ] 주문 생성 시 **주문 상태는 "대기"** 여야 한다.
- [ ] 주문 타입이 **배달인 경우, 배달 주소는 필수값**이다.
- [ ] 주문 타입이 **먹고가기인 경우, 주문 테이블이 존재해야 하며 비어 있어야 한다**.

### 주문 접수
- [ ] 주문이 존재해야 한다.
- [ ] 현재 주문 상태가 "대기"여야 한다.
- [ ] 주문 타입이 **배달인 경우, 배달 접수 처리를 해야 한다**.

### 주문 서빙
- [ ] 주문이 존재해야 한다.
- [ ] 현재 주문 상태가 "접수"여야 한다.

### 주문 배달
- [ ] 주문이 존재해야 한다.
- [ ] 주문 타입이 "배달"이어야 한다.
- [ ] 현재 주문 상태가 "서빙"여야 한다.

### 주문 배달 완료
- [ ] 주문이 존재해야 한다.
- [ ] 주문 타입이 "배달"이어야 한다.
- [ ] 현재 주문 상태가 "배달중"이어야 한다.

### 주문 최종 완료
- [ ] 주문이 존재해야 한다.
- [ ] 주문 타입이 **배달인 경우, 현재 주문 상태가 "배달 완료"여야 한다**.
- [ ] 주문 타입이 **포장 또는 먹고가기인 경우, 현재 주문 상태가 "서빙"여야 한다**.
- [ ] 주문 타입이 **먹고가기인 경우, 주문 테이블을 비워야 한다**.

### 주문 조회
- [ ] 모든 주문을 조회할 수 있어야 한다.

---

## 5. 주문 테이블 관리

### 주문 테이블 생성
- [ ] 주문 테이블명은 필수값이다.
- [ ] 손님 수는 **0으로 초기 설정**한다.
- [ ] 자리 상태는 **비움으로 초기 설정**한다.

### 주문 테이블 착석
- [ ] 주문 테이블이 존재해야 한다.
- [ ] 자리 상태를 **만석으로 변경**해야 한다.

### 주문 테이블 비우기
- [ ] 주문 테이블이 존재해야 한다.
- [ ] 손님 수를 **0으로 설정**해야 한다.
- [ ] 자리 상태를 **비움으로 변경**해야 한다.
- [ ] 주문 상태가 **완료된 상태**여야 한다.

### 주문 테이블 손님 수 변경
- [ ] 손님 수는 0보다 커야 한다.
- [ ] 주문 테이블이 존재해야 한다.
- [ ] 자리 상태가 **만석 상태**여야 한다.

### 주문 테이블 조회
- [ ] 모든 주문 테이블을 조회할 수 있어야 한다.
